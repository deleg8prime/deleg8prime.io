# deleg8 - Virtual Assistant Agency Website

A modern, professional website for **deleg8** (Delegate), a virtual assistant and task delegation agency.

## Features

- **Responsive Design**: Works beautifully on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional interface with smooth animations
- **Service Showcase**: 6 core services displayed with icons
- **Team Section**: Meet the professional team members
- **Testimonials**: Client success stories with 5-star ratings
- **Contact Form**: Easy way for potential clients to reach out
- **Smooth Navigation**: Smooth scrolling between sections
- **Interactive Elements**: Hover effects and animations throughout

## Files Included

- `index.html` - Main website file with structure and content
- `styles.css` - Complete styling and responsive design
- `script.js` - Interactive features and animations
- `README.md` - This file

## Quick Start

1. Open `index.html` in your web browser
2. Navigate through the website using the top menu
3. Fill out the contact form to get started
